ARRAYS: Estos nos permiten almacenar elementos (de cualquier tipo de datos) tantos como queramos y acceder a ellos con un solo nombre, el mismo con el que se definió el array al momento de su creación. Para acceder a los distintos elementos dentro de él se utiliza el nombre del array seguido por un número (índice) que indica la posición del elemento que estamos buscando, siempre la primera posición del array tiene el índice 0. El mayor índice será igual a n-1, siendo n el total de elementos contenidos en el array.
